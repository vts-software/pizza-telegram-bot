# Pizza Telegram Bot

Telegram bot for ordering pizza built with Django and PostgreSQL.

## Project Description

This project is a backend system and Telegram bot that allows users to browse a pizza menu, add pizzas to a cart, and create orders.

The system includes:

- Django backend
- PostgreSQL database
- Telegram bot interface
- REST API
- Admin panel for managing menu and orders
- Automated tests

## Technologies

- Python
- Django
- PostgreSQL
- Telebot
- Django ORM
- Git

## Features

Users can:

- View pizza menu
- Add pizzas to cart
- View cart
- Create orders
- View order history

Admin can:

- Add new pizzas
- Edit menu
- Manage orders
- Manage users
